Drop-down List IBGE - Cidades e Estados

O objetivo deste script é buscar de forma assíncrona listas de estados e cidades brasileiras através da API do IBGE e exibí-los em elementos SELECT do HTML de forma dinâmica.
Autor: Eduardo Vicenzi Kuhn

Exemplo 1

Cidades por estados (utilizando SIGLA como value para estado,
e NOME como value para cidade)

Ex.: <option value="RS">Rio Grande do Sul</option>
<option value="Porto Alegre">Porto Alegre</option>

Exemplo 2

Cidades por estados (utilizando NOME como value para estado,
e ID como value para cidade)

Ex.: <option value="Paraná">Paraná</option>
<option value="4106902">Curitiba</option>

Exemplo 3

Cidades por estado pré-definido (neste exemplo: 'RS')


Implementação

Estados:

selectEstados(id_elemento, tipo_value);
Sendo:
string id_elemento = ID do elemento HTML select dos estados
string tipo_value(opcional) = ID, SIGLA(padrão) ou NOME

Cidades:

selectCidades(id_elemento, uf, uf_select, tipo_value)
Sendo:
string id_elemento = ID do elemento HTML select das cidades
string uf(opcional) = nome, id, ou sigla do estado, ou id do elemento select dos estados
boolean uf_select(opcional) = caso true, irá utilizar o select de estados (uf) para obter cidades
string tipo_value(opcional) = ID, SIGLA(padrão) ou NOME
